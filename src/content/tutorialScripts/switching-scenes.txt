A scene, is a folder in the game project, containing "main" file and optional additional scripts and UI designers.
A scene usually used for organizing and running everything required for a specific game level or game stage.
```
Game       - [g]Intro
Game level - [y]one
Game level - [m]Train
``` 
[img2: /assets/tutorials/physics-gameplay/pasted-2026-06-23T17-33-38-984Z.png]
for example, in this fighting game folder structure, we can see how the game levels are organized. Each game level has its own folder, its own level UI designer, its own "main" file, and additional files and folders as needed.
[wait: 2]
```
canvas.size 1600,900
screen.mode borderless 

switch to [circle: "\"game_intro\"", color: r]
```
The general "main" file of the game, decides which game level to start with. In the demo fighting game, we start with the scene called "game_intro", which is placed in a folder with exactly that name as you can see in the folder structure picture. The "switch to" command, tells the game engine to run the "main" file in the specified folder.

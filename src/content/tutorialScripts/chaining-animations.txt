```
player=>Ninja
player.Kick then Backflip then "Spin" then Block then SideKick
```
SceneMax animation controller can play several model animations in 1 command. In this example the Ninja player does several martial arts moves including attacking and blocking.
You might wonder why the **Spin** animation is surrounded by double-quates? this is because the word **Spin** is also a key-word in the SceneMax language.  
```
player=>Ninja
```
SceneMax also allows you to control the animation speed
```+
player.Kick at speed of 0.5 
```
so for example in this code the **Kick** animation is performed two times slower than the original speed.
```+
  then Backflip
```
```+
  then "Spin" at speed of 2 
```
the **Spin** animation is played two times faster
```+
  then Block 
  then SideKick at speed of 0.1
```
and the **SideKick** is played 10 times slower than the original speed.

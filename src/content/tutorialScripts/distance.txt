Distance explains how one named value is stored, read, or updated.
Use this when one value needs a meaningful name. It is better than repeating the same number or text in several commands.
In a fighting game, language fundamentals track health, score, current action, input state, and whether a hit is allowed.
The sample begins with var dist = distance(n,n2). In plain language, it gives a value a name so later commands can read or change it.
The key idea is meaning: the name explains what the value represents in the game.

Sometimes the effect should follow an object, but not from the object's exact center.
Add a pos offset to the attach command. The offset is local to the parent, so it moves with the parent.
In the sample, aura_fx attaches to player with pos zero, one, zero. That places the effect one unit above the model while still following it.

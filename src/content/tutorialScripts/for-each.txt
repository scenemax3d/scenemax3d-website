For each runs the same command on many objects.
Instead of naming every model one by one, SceneMax gives you each matching object through a temporary variable.
In this example, m represents the current model in the loop.
The turn command runs once for Sinbad and once for Ninja.
Use for each for cleanup, highlighting, resetting enemies, or applying the same effect to all objects of one type.

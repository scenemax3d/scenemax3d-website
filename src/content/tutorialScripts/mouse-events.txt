A mouse event runs code whenever the player clicks with the mouse on some game object.
```
player1 => sinbad

when [g]mouse [g]left is pressed on [y]player1 do
  player1.turn left 30 in 1 second
end do
```
in this example, the player1 game object, will turn 30 degrees to the left whenever the user clicks with the mouse left button on it. In Android, this will translate to the user, touching player1.

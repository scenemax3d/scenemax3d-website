```
player => Ninja
player.Kick async
```
at any given time a model animation can be played for example **player.Kick**.
```+
wait 0.1 seconds
player.BackFlip
```
while the animation is played, there is a chance that it will be cancelled in case a different animation is triggered. This is what we call - **animation cancelling** and it allows a more fast, fluent game where actions (animations) can be switched before ending.
```
player=>Ninja 
player.Death1 at speed of 0.2 : protected Async
wait 0.2 second 
player.Kick 
```
However, there are times when we would like to **protect** the animation from being cancelled by another animation. 
```#
[mod: "protected", color:00FF00]
```
In that case we put the **protected** keyword in the end of the animation command. 
In this example, the **Kick** animation will not cancel the previous **Death1** animation.

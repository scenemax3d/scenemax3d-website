A returning-value pattern lets one block calculate a result and give that result back to the caller.
The documented return command is meant for that job, but this runtime-safe sample uses an output variable because the current runtime has trouble with return blocks.
Here, get_bonus calculates the bonus by setting score_bonus to 25.
The caller runs get_bonus, then adds score_bonus to the score.
Use this pattern for scoring, rewards, calculations, and other small results that one reusable block should produce.

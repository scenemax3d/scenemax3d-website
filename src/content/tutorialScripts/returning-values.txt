A procedure which returns a values is called a function. In this example the function is called "get_bonus" and it is returns a fixed values of 100 which is printed on the screen.
```
var score = 10

get_bonus = {
	return 100
}

score = score + get_bonus()
sys.print score
```
A function return value can also be added to an existing variable as shown in the above code sample.

SceneMax provides us with a special model property named "anim_percent", which can be read whenever we like to examine the current animation render percent.
```
if (player.anim_percent > 35 && player.anim_percent < 45) {
  // allow the punch to take effect
}
```
Sometimes, it is useful to examine the current animation percent of a given model in order to control the flow of the game. For example, when a fighter player kicks, the effective part of the kick might be in the range of 35 to 45 percent. only in that range we allow the opponent to actually get hit in case the player kicks him. The code above demonstrates a typical animation percent check.

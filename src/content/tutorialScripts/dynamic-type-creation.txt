Dynamic object type is the ability to determine the type based on some logic.
```
var [r]object_types = ["Sinbad", "Ninja", "Dragon"]
```
in this example, we put some possible object types in an array of strings.
```+
[r]my_model => (object_types[ rnd(3) ])
```
then, we create a model, where its type is randomly selected from the **object_types**  array.
```+
my_model.[r]turn left 360 in 5 seconds
```
finally, we turn the created object, left 360 degrees.

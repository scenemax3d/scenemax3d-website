SceneMax3D has a default camera object placed in the scene. The camera's position and point of view determines what the player sees on the screen at any given time.
The camera can be modified in run-time.
```
s => sinbad
camera.[color:00FF00]chase[color:default] s
```
In this example, the camera follows (chases) the "s" game object. So whenever the object moves, the camera will follow it (by default from behind).
```
camera.[g]pos (10,10,5)
```
camera can be placed in a random position in 3D space.
[wait: 2]
```
camera.[g]turn left 360 in 10 seconds
```
camera can be turned to any direction like any other game object

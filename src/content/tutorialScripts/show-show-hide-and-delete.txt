Show makes a hidden effect visible again.
The full beginner pattern is: create and play the effect, hide it when it should disappear, then show it when it should return.
This is useful when an effect object is already created and you want to reuse it without declaring it again. For example, a shield aura can be hidden when the shield is off, then shown again when the player activates the shield.

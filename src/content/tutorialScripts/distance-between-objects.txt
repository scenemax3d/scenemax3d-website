Distance Between Objects explains one basic movement, rotation, facing, or measurement idea.
Use this when position or facing direction affects gameplay, not just appearance.
In a fighting game, this is what makes a punch, jump, dodge, replay path, or knockback feel responsive instead of random.
The sample begins with n => ninja : pos (-2,0,0). In plain language, it creates or loads an object and gives it a name for later commands.
The key idea is to know what problem this feature solves before using it in a larger scene.

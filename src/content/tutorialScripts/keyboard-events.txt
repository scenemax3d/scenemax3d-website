A keyboard event runs code when the player presses a key.
The event block starts with when key, the key name, and the phrase is pressed.
Adding once means the action happens once per press instead of repeating every frame while the key is held.
In this example, pressing the up key moves the player forward.
Use keyboard events for movement, attacks, jumps, menu shortcuts, and testing gameplay quickly.

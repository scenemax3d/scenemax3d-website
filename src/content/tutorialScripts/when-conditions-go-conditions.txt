A when condition waits for something to become true.
This is different from if, which checks only at the moment the if line runs.
In this example, the when block watches door_open. When the value becomes 1, the message runs.
Use when conditions for doors, puzzle triggers, timers, enemy defeat checks, and story events.
They help scripts react to gameplay changes without constantly writing manual checks.

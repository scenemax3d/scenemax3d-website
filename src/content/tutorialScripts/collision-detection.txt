Use collision detection event handler for responding to collision between two moving objects.
```
b1=>box: pos (-4,0,0)
b2=>box: pos (4,0,0)
```
Let's take for example two boxes
```+
b1.move left 8 in 5 seconds async
b2.move right 8 in 3 seconds async
```
Moving one against the other.
```+
when b1 collides with b2 do
  sys.print "BOOM!"
end do
```
we will add a collision detection event handler to do something when they collide with each other while moving. In this case we simply print the word "Boom!".

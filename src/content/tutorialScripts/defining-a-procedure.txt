A procedure is a named block of commands.
Define a procedure when you want to reuse the same behavior in several places.
In this example, spin_box stores one turn command for the box.
The procedure definition itself does not run the turn yet. It only creates the reusable command block.
Use procedures for repeated effects, enemy actions, UI updates, level setup, and cleanup steps.

A numeric variable stores a number.
Use numeric variables for score, health, ammo, timers, speed, level number, or damage.
The line var score = 0 creates the variable and gives it a starting value.
After that, the script can add to it, subtract from it, compare it, or print it.
Numeric variables let the game remember changing numbers while it runs.

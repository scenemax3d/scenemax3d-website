Resource Declaration explains a helper for hits, attachments, resources, or debugging around objects.
Use this when visual objects also need gameplay meaning, such as being hit, attached, detected, or debugged.
In a fighting game, objects and assets become fighters, weapons, hit colliders, arena pieces, pickups, and visual markers.
The sample begins with using ninja,sinbad,dragon model. In plain language, it shows the command or data shape used by this lesson.
The key idea is to know what problem this feature solves before using it in a larger scene.

SceneMax3D has a few built-in, run-time only, particle effects.
```
Flash
Explosion
Debris
Spark
Smoketrail
Shockwave
TimeOrbit
Flame
```
[wait: 2]
In runtime, at any given time, you can play a built-in particle effect for example:
```
s => Sinbad
effects.explosion.show : pos (s)
```
this code will show the "explosion" particle effect in the position if the **Sinbad** model called "S".
[wait: 1]
```
effects.flame.show : pos (13,-15,43) and radius = 5 and duration 10
```
In the same way, 
```#
[mod: "radius", color:00FF00]
```
we can control the radius in meters
```#
[mod: "duration", color:00FF00]
```
and the duration in seconds, of the played effect.
[wait:3]
```
effects.TimeOrbit.show : pos (0,0,0) and [color:00FF00]end size = 7[color:default] and duration = 10
```
we can also control the start-size and the end-size of the effect's particle system.

Basic Replay explains how designed or recorded movement plays back consistently.
Use this when a movement path should repeat exactly, like a recorded camera move, entrance, or scripted object motion.
In a fighting game, this is what makes a punch, jump, dodge, replay path, or knockback feel responsive instead of random.
The sample begins with var points = [0,0,0,0,0,0, 1,1,10,0,90,0, 2,2,0,0,0,0]. In plain language, it creates one named list instead of many separate variables.
The key idea is to know what problem this feature solves before using it in a larger scene.

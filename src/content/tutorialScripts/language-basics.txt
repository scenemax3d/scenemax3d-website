[video: /assets/tutorials/getting-started/07-package-saved-game-executable.mp4]
When the main file is saved and the scene runs, package the folder as a desktop executable.
Select the scene folder, use Package Program, keep Windows selected, and click Package.
SceneMax reports the generated output path when the package is finished.
[video: /assets/tutorials/getting-started/08-run-saved-game-executable.mp4]
Open the output folder, go into windows, and run the generated executable.
This is the final beginner loop: save the project, package the scene, then launch the saved game outside the IDE.

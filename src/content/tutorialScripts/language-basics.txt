In order to pack your game and prepare it for deployment to the marketplace, you need to be focused on the "main" script file and then click on the "Package program" button. This will open the package and deployment dialog. 
[video: /assets/tutorials/getting-started/07-package-saved-game-executable.mp4]
It should take less than a minute for the package process to finish creating the game executable. Alongside the game file, SceneMax creates a report about all the assets being used in the packed game so you can try and fine-tune it and reduce its size if needed.

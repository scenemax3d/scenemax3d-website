A guard condition is a gate placed before a procedure, event, or action.
The command below the condition can run only when the condition is true.
[wait:1]
In this sample, pressing on "X" will do something 
```#
[clear]
[mod: "player1_ko==0",color:FF0000]
```
only if the player is not knocked-out, 
```#
[clear]
[mod: "game_status!=GAME_STATE_OVER",color:FF0000]
```
and the game is not over.
[wait:2]

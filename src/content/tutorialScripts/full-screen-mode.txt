In your project's main file, you can set the screen size and mode.
```
canvas.size 1600,900
screen.[g]mode borderless 
```
for example, when setting the screen mode to "borderless", the game will render on a flat screen with no borders. exiting the game will be done by pressing the **Escape** button.
```
canvas.size 1600,900
screen.[g]mode full
```
screen mode full, is the most impressive mode. All the screen is used for rendering the game's scene.

Running a procedure means executing the commands saved inside it.
Use the word run followed by the procedure name.
Here, spin_box is defined first, then run spin_box starts the box turn.
This keeps the action readable because the event can say what it wants to do instead of repeating every command.
Use run when a named behavior should happen now.

A shared variable is kept across levels.
Use shared when the value belongs to the whole game, not only the current scene.
In this example, score is shared, so it can continue from one level to the next.
This is useful for total score, player lives, unlocked items, game settings, and campaign progress.
Do not use shared for temporary values that should reset when a level reloads.

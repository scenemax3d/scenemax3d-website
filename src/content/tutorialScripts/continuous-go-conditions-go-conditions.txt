A continuous go condition starts with the # symbol.
Regular go conditions check before a command starts. Continuous go conditions keep checking while the block runs.
In this example, the movement loop is allowed only while player_is_jumping is 0.
If another part of the game changes player_is_jumping, the loop can stop immediately.
Use this for movement, attacks, charging actions, or any behavior that must cancel when the game state changes.

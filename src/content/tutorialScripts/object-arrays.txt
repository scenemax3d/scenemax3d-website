An object array stores actual scene objects, not just numbers or names.
This is helpful after objects already exist and you want one rule to work on any of them.
Here, my_objects contains Sinbad, Ninja, a sprite, and a box. random_obj receives one real object from that list.
Because random_obj is an object reference, the next command can turn the selected object directly.
Use object arrays for enemy groups, pickup choices, targets, patrol objects, or anything where the game should choose from existing objects.

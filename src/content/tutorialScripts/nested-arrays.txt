A nested array is an array that contains other arrays.
Use nested arrays when your data has groups inside groups, such as waves inside a level or rows inside a grid.
In this example, easy_wave and hard_wave are two separate enemy lists. waves stores both lists together.
The line waves[0] selects the first wave, so the game can decide which group of enemies to spawn next.
Nested arrays keep related lists organized instead of scattering many variable names across the script.

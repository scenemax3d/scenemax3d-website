Effekseer is a popular open source, particle effect editing tool. SceneMax3D has a built-in Effekseer effects importer. 
[img: /assets/tutorials/lighting-visuals/pasted-2026-06-22T18-10-35-596Z.png]
Simply select Assets -> import Effekseer effect, select the Effekseer file which can be downloaded from their website, give an ID and that's it.
```
laser_effect => effects.effekseer.[circle: "Homing_Laser01_3", color r]
laser_effect.play pos (player1)
```
[img2: /assets/tutorials/lighting-visuals/pasted-2026-06-22T18-21-07-109Z.png]
in run-time, simply load the Effekseer effect and play it. in this sample, we load the "Homing_Laser01_3" effect and play it in the position of player1.

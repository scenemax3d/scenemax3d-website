A SceneMax project is the saved workspace for one game. Start in File, open Projects, choose New Project, type a short project name, and confirm it.
SceneMax creates the project folder, adds the default scripts area, and switches the IDE title to the new project.
[video: /assets/tutorials/getting-started/02-open-existing-project-flow.mp4]
To return to work later, open File, Projects, and select an existing project from the list.
This is the project operation loop beginners need first: create a project, switch projects deliberately, and always check the title bar before adding files.

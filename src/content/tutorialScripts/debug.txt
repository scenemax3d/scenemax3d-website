Debug mode helps you inspect what is happening while building the game.
Turn it on when you need extra information for testing.
Turn it off when you want the normal player view again.
In this example, debug mode is on only while the developer is checking the scene.
Use debug for collisions, object placement, camera problems, and other issues that are easier to solve when you can see extra runtime information.
